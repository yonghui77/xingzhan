# 开发路线

## 当前：0.40 星港大厅与中央舰船展示版本

- 4 个安全等级不同的星系
- EVE/交易所风格地区市场、订单深度、价格曲线与价差
- 价格走势图支持鼠标悬停查看周期、价格、涨跌幅和库存
- 玩家可自定义出售价格：可立即吃买盘，也可生成本站限价卖单
- 玩家卖单会真实锁定本地货物，可在“我的挂单”中等待撮合或撤单
- 星港中枢重构为大厅式进站界面：左上玩家身份卡、中间伪 3D 舰船、右侧功能区和右下离站按钮
- 当前舰船展示使用程序化 HTML/CSS 模型，无需额外建模资源；后续可升级为 Three.js / glTF 真 3D
- 模块按钮加入玻璃拟态、轻按缩放、回弹高光和更明确的点击反馈
- 8/16/32/64 名 AI 玩家自主采矿、贸易、探索与护航
- 动态威胁预算、四类敌舰和海盗据点战
- 统一设置面板：字号、字体、画质和中英文
- 独立主音量、音乐和音效控制
- 程序化背景音乐和完整游戏操作音效
- 进入空间站默认展示星港中枢，而不是直接打开市场
- 星港模块：市场交易所、舰船维护区、任务大厅、仓库货舱、星图导航、情报终端、系统设置
- 站内机库视觉、停泊飞船、模块点击反馈和动态空间站状态
- 物品实体化：飞船货舱、空间站仓库、市场库存彼此独立
- 空间站市场下拉查看：可查看全星系物价，但只能在当前停靠站交易
- 本站仓库存取与基础制作工坊
- 2.5D 视觉层：空间站透视、面板层次、星空视差和三档 3D 视觉设置
- 玩家舰船“拓荒者级”模型：艇身、座舱、收敛侧翼、双引擎、推进尾焰和护盾可视化
- “边境飞行员执照”新手引导：停靠、市场观察、采矿、本地交易、远程行情、独立仓库和自由合约
- 合约中心“边境档案”主线面板：章节目标只读取玩家真实行为，不直接改价、刷库存或操控税率
- 星港中枢“主线信标”：展示当前章节、下一步目标和跳转入口，但不替玩家计算利润或执行交易
- 修复真实窗口下空间站中枢高度溢出、舰船视窗过高和右侧模块滚动不稳定问题
- 修复星港停靠舰船模型：左右翼收窄并重新分层，避免大三角破面、穿过机库环或左右不对称
- 主线目标从“教程完成”改为“真实行为完成”：停靠、采矿、出售、远程行情、合约、改装、跨星系访问和 PVE 记录
- 新增剧情/经济边界测试：渲染主线和查看远程行情不会改变市场状态
- 科技感 UI 强化：全局扫描网格、HUD 角标、玻璃终端材质、霓虹分割线、面板扫描光和统一按钮反馈

## 下一阶段：经济生产链

- 矿石精炼、组件制造和舰船生产
- 玩家买单、订单修改费用和更完整的成交量统计
- 运输合同和货物保险
- 战争需求与 NPC 限额采购
- 商品成交量和更完整的价格历史

## 联机阶段

- 服务端权威存档和市场订单簿
- 玩家组织、空间站税率与设施维护
- 多人同星系、护航和自由交战
- 反作弊、保险套利识别和经济监控后台
