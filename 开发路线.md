# 开发路线

## 当前：0.51 可玩性与信息密度优化版本

- 舰船维护拆分为舰体、装配、制造、升级四个轻量视图
- 舰体建造信息、插件参数和制作材料默认折叠，保留完整详情但减少首屏文字
- 四舰体加入统一 3D 概念渲染缩略图和船坞主预览
- 价格走势图加入鼠标与键盘双通道数据提示
- 建立新手、交易者、工业玩家、战斗玩家四类自动化体验评分
- 新增拓荒者、信使、开拓者、堡垒四种舰体定位与独立基础性能
- 新舰体由当前空间站使用本地材料和 ISK 建造，闲置舰船保留真实停放位置
- 每艘舰船独立保存升级、插件与当前武器配置
- 四种舰体拥有不同 Canvas 飞行轮廓、机库 2.5D 模型、颜色、尺寸和引擎布局
- 船坞支持预览未拥有舰体、查看建造材料、异地停放状态和本站切换
- 新增探索者、工业家、贸易商、执法者四条长期路线，所有活动进度并行累计
- 追踪路线只改变 HUD 当前目标；完成路线获得不可交易身份徽章，不向市场注入商品
- 飞行员执照完成后，教程卡自动切换为长期职业追踪器
- 插件制造界面加入安装前数值模拟，并与真实舰船数值共用同一计算逻辑
- 拟合预览明确标记提升、代价与不变项，减少玩家反复制造试错
- `E` 交互改为动态情境按钮，实时显示停靠、回收、采矿、距离和进度，并支持鼠标点击
- 修正矿脉与战利品靠近时提示动作和实际触发优先级不一致的问题
- 射击模式加入独立可点击入口，键盘操作增加按压高亮，输入框获得焦点时屏蔽飞行快捷键
- 战斗中自动收纳教程、合约、货舱与日志，只保留直接影响生存和瞄准的情报
- 左侧信息栏保留边缘唤回区，鼠标悬停即可临时展开，不打断战斗
- 武器新增均衡、精准、连射模式，并真实改变伤害、射速、散布、电容与热量
- 4 个安全等级不同的星系
- EVE/交易所风格地区市场、订单深度、价格曲线与价差
- 价格走势图支持鼠标悬停查看周期、价格、涨跌幅和库存
- 玩家可自定义出售价格：可立即吃买盘，也可生成本站限价卖单
- 玩家卖单会真实锁定本地货物，可在“我的挂单”中等待撮合或撤单
- 星港中枢重构为大厅式进站界面：左上玩家身份卡、中间伪 3D 舰船、右侧功能区和右下离站按钮
- 星港中枢清爽化：交易行、改装、仓库、任务、星图、情报、设置排为顶部一行
- 主线目标和合约收纳到“任务”入口，避免进站大厅文字溢出或被卡住
- 离站按钮缩小为右下角轻量 CTA，不再占据右侧整块面积
- 船坞主展示使用高精度概念渲染；实战模型继续使用程序化 HTML/CSS/Canvas，后续可升级为 Three.js / glTF 真 3D
- 模块按钮加入玻璃拟态、轻按缩放、回弹高光和更明确的点击反馈
- 舰船插件系统第一版：武器、防御、工程、工业、电子五类槽位
- 插件需要通过材料制造，并存放在当前空间站插件库存中
- 新增材料链：钛合金、纳米陶瓷、超导纤维、量子透镜、等离子电容、制导芯片
- 新增武器差异：磁轨炮稳定实弹、激光高精度高发热、导弹高爆发且消耗弹药
- 敌舰采用护盾与装甲双层耐久：热能破盾、动能破甲、爆炸伤害覆盖小范围目标
- 目标 HUD 显示双层耐久、武器伤害倍率与命中伤害数字
- 地区制造专精上线：工业区、矿业边境和战争前线对指定材料提供额外本地产出
- 舰船盾甲状态改为底部中央同心圆：外圈护盾、内圈装甲，并保留燃料与货舱简报
- 射击、交互、推进和星图按钮分列圆环左右，左上大型舰船卡取消
- 盾甲进入低值时自动切换“护盾低 / 危险”状态和动态警示
- 11 类市场商品拥有独立 3D 物品图，市场列表改为缩略图、名称、价格和涨跌幅
- 商品长说明与库存/持仓细节退出常驻列表，改为悬停信息和详情数据
- 制作配方与插件蓝图加入一键采购缺失材料，报价包含本站实时价格和交易税
- 采购采用整单校验，不允许远程市场购买，不足余额或本站缺货时不会产生部分成交
- 全局 UI 改为深色玻璃、低描边、大圆角和弹性按压的简约体系
- 空间站标签采用胶囊选择状态，功能入口隐藏重复说明文字
- 基础脉冲炮与已安装武器支持 Q 键/点击快速切换
- 武器电容、射击热量、过热锁定和自动冷却机制上线
- 激光、磁轨炮和导弹拥有不同电容消耗、发热速度、弹药需求与伤害类型
- 敌舰弹药接入热能、动能和爆炸伤害，玩家护盾与装甲承受不同倍率
- 插件真实影响舰船数值：火力、护盾、装甲、速度、货舱、采矿和弹道表现
- 8/16/32/64 名 AI 玩家自主采矿、贸易、探索与护航
- 动态威胁预算、四类敌舰和海盗据点战
- 统一设置面板：字号、字体、画质和中英文
- 独立主音量、音乐和音效控制
- 程序化背景音乐和完整游戏操作音效
- 进入空间站默认展示星港中枢，而不是直接打开市场
- 星港模块：市场交易所、舰船维护区、任务大厅、仓库货舱、星图导航、情报终端、系统设置
- 站内机库视觉、停泊飞船、模块点击反馈和动态空间站状态
- 物品实体化：飞船货舱、空间站仓库、市场库存彼此独立
- 空间站市场下拉查看：可查看全星系物价，但只能在当前停靠站交易
- 本站仓库存取与基础制作工坊
- 2.5D 视觉层：空间站透视、面板层次、星空视差和三档 3D 视觉设置
- 玩家舰船“拓荒者级”模型：艇身、座舱、收敛侧翼、双引擎、推进尾焰和护盾可视化
- “边境飞行员执照”新手引导：停靠、市场观察、采矿、本地交易、远程行情、独立仓库和自由合约
- 合约中心“边境档案”主线面板：章节目标只读取玩家真实行为，不直接改价、刷库存或操控税率
- 星港中枢“主线信标”：展示当前章节、下一步目标和跳转入口，但不替玩家计算利润或执行交易
- 修复真实窗口下空间站中枢高度溢出、舰船视窗过高和右侧模块滚动不稳定问题
- 修复星港停靠舰船模型：左右翼收窄并重新分层，避免大三角破面、穿过机库环或左右不对称
- 主线目标从“教程完成”改为“真实行为完成”：停靠、采矿、出售、远程行情、合约、改装、跨星系访问和 PVE 记录
- 新增剧情/经济边界测试：渲染主线和查看远程行情不会改变市场状态
- 科技感 UI 强化：全局扫描网格、HUD 角标、玻璃终端材质、霓虹分割线、面板扫描光和统一按钮反馈
