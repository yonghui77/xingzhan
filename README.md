# 星际商路：边境纪元 0.32

以地区市场、风险运输、即时战斗、舰船成长和动态 PVE 为核心的浏览器游戏。

## 启动

直接双击 `index.html`，或通过 GitHub Pages 访问。

## 操作

- `WASD`：移动
- 鼠标：瞄准
- 鼠标左键 / `空格`：射击
- `E`：采矿、拾取、停靠
- `Shift`：推进
- `M`：星图
- `Esc`：关闭面板

## 当前重点

- 进入空间站后默认展示“星港中枢”，不再直接打开市场。
- 星港中枢包含机库视觉、舰船停泊、空间站状态与模块入口。
- 市场、舰船维护、任务大厅、仓库货舱、星图导航、情报终端和系统设置都作为站内模块进入。
- 顶部状态会随模块切换显示“星港中枢 / 市场终端 / 舰船维护”等。

## 已实现

- 4 个不同安全等级和经济特性的星系
- 地区独立价格、库存、价格走势和买卖盘深度
- 8/16/32/64 名可配置 AI 玩家
- AI 矿工、商人、探索者和雇佣兵自主活动并影响市场
- 采矿、战斗、战利品、运输和交易
- 动态威胁预算、四类敌舰、警戒增援和海盗据点
- 护盾可视化、舰船升级、损毁保险和本地存档
- 统一游戏设置面板
- 标准、大号、特大三档界面字号
- 清晰、圆润、终端三种字体风格
- 流畅、均衡、精细三档画面质量
- 简体中文与 English 核心界面
- 主音量、背景音乐和游戏音效独立控制
- 程序化太空氛围音乐
- 射击、护盾、装甲、采矿、交易、跃迁、警戒和爆炸音效
