# 星际商路：边境纪元 0.3

以地区市场、风险运输、即时战斗和舰船成长为核心的浏览器游戏垂直切片。

## 启动

直接双击 `index.html`。

## 操作

- `WASD`：移动
- 鼠标：瞄准
- 鼠标左键 / `空格`：射击
- `E`：采矿、拾取、停靠
- `Shift`：推进
- `M`：星图
- `Esc`：关闭面板

## 已实现

- 4 个不同安全等级和经济特性的星系
- 地区独立价格、库存、价格走势和买卖盘深度
- 8/16/32/64 名可配置 AI 玩家
- AI 矿工、商人、探索者和雇佣兵自主活动并影响市场
- 采矿、战斗、战利品、运输和交易
- 锁定式持续采矿束、距离中断和采集进度
- 护盾容量环、舰船护盾场和受击方向反馈
- 动态威胁预算与视野外安全刷新
- 侦测、锁定、武器三层作战范围
- 侦察无人机、截击舰、导弹舰和重型炮舰
- 采矿工业信号、海盗伏击与四级警戒增援
- 分阶段海盗据点：护盾、核心、防御舰队和地区奖励
- 海盗活动影响 AI 运输损失与当地补给价格
- 空间站停靠、维修、舰船升级
- 动态合约和成长奖励
- 舰船损毁、有限保险和本地存档
