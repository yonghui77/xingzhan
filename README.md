# 星际商路：边境纪元 0.51

以地区市场、风险运输、即时战斗、舰船成长和动态 PVE 为核心的浏览器游戏。

## 启动

直接双击 `index.html`，或通过 GitHub Pages 访问。

## 操作

- `WASD`：移动
- 鼠标：瞄准
- 鼠标左键 / `空格`：射击
- `Q`：切换基础武器与已安装武器
- `R`：切换均衡、精准与连射模式
- `E`：采矿、拾取、停靠
- `Shift`：推进
- `M`：星图
- `Esc`：关闭面板

## 当前重点

- 舰船维护拆分为“舰体 / 装配 / 制造 / 升级”四个独立视图，避免所有系统同时堆在一个长页面。
- 舰体、插件和基础配方采用渐进披露：首屏保留名称、定位、产物与操作，参数、材料和产线说明点击展开。
- 四类舰体接入统一风格高精度概念渲染，船坞主预览与列表缩略图共用同一视觉语言；实战仍使用实时程序化模型。
- 市场价格图支持鼠标悬停、键盘聚焦和方向键逐周期查看价格、涨跌与库存。
- 新增四类自动化模拟玩家测试：新手、交易者、工业玩家、战斗玩家，覆盖 20 项关键体验条件。
- 新增拓荒者、信使、开拓者、堡垒四种舰体，分别对应均衡、高速运输、工业采矿和重型战斗。
- 每种舰体拥有独立护盾、装甲、火力、货舱、航速、采矿、电容、燃料和碰撞尺寸。
- 新舰需要在当前空间站使用本地材料与 ISK 建造；未启用舰船真实停放在建造或换船所在空间站。
- 每艘舰船独立保存插件、升级和武器配置，切换舰船不会复制或丢失装备。
- 飞行 Canvas、空间站机库、改装预览和圆形 HUD 均会显示对应舰体的独立轮廓、颜色与引擎布局。
- 新增探索者、工业家、贸易商、执法者四条职业路线；所有路线同时累计，玩家只是在选择当前追踪目标。
- 职业完成奖励是不可交易的身份徽章，不直接发放商品、不控制价格，保持沙盒经济独立。
- 新手执照完成后，左侧教程卡会自动转换为长期职业追踪器。
- 插件蓝图加入拟合模拟，在制造和安装前显示火力、射速、热量、盾甲、速度、货舱、采矿或弹道变化。
- 拟合模拟与真实安装共用同一套舰船数值计算，避免界面预览与实际效果不一致。
- 飞行交互改为情境按钮：靠近空间站、矿脉或战利品时，`E` 按钮会同步显示实际动作、距离和关键状态，也可以直接点击。
- 修正矿脉与战利品重叠时提示和实际动作不一致的问题；采矿进行中再次交互始终优先关闭采矿束。
- 射击模式增加独立的鼠标点击入口，Q/E/R/Shift/M 按键均有即时按压反馈；在输入框打字不会误触飞行快捷键。
- 战斗 HUD 会在敌人进入交战状态后自动收纳非必要信息，只保留目标、雷达、威胁与舰船资源；悬停左侧边缘可临时展开。
- 武器加入均衡、精准、连射三种模式，分别改变伤害、射速、散布、电容与热量消耗。
- 物品真实存在于所在地：飞船货舱、各空间站仓库、市场库存彼此独立。
- 每个空间站都可以交易，但只能在当前停靠站实际买卖；远程市场只能通过下拉菜单查看行情。
- 市场界面升级为交易所风格：商品列表、价格走势、市场深度、玩家成交动态、地区对比和持仓/挂单信息并列展示。
- 价格走势图支持鼠标悬停查看周期、价格、涨跌幅和当前库存，便于玩家自行判断倒卖价值。
- 出售不再只能按系统即时价成交：玩家可以自定义出售单价，低于/等于买盘时立即成交，高于买盘时形成本站限价卖单。
- 玩家卖单会真实锁定本地货物，并显示在“我的挂单”中，可等待市场撮合或手动撤单。
- 购买便宜货物必须驾驶战舰前往对应空间站，装入货舱后再运输到其他站点出售。
- 每个空间站拥有独立仓库，可存入、取出物品；基础制作产物会进入当前站仓库。
- 进入空间站后默认展示“星港中枢”，不再直接打开市场。
- 星港中枢升级为大厅式进站界面：左上玩家身份卡、中间伪 3D 舰船机库、右侧功能入口和右下大号离站按钮。
- 星港中枢 0.41 清爽化：交易行、改装、仓库、任务、星图、情报、设置收进顶部功能条。
- 主线目标和合约不再铺开占位，统一收纳到“任务”入口，避免文字在进站大厅被卡住。
- 离站按钮缩小为右下角轻量 CTA，不再占据整块右侧区域。
- 船坞使用高精度概念渲染展示舰体质感；飞行、护盾和即时状态仍由 HTML/CSS/Canvas 程序化生成，后续可替换为 Three.js / glTF 真 3D 模型。
- 功能按钮加入类 Apple 的玻璃质感、按压缩放、回弹高光和点击音效反馈。
- 星港中枢包含机库视觉、舰船停泊、空间站状态与模块入口。
- 舰船插件系统第一版上线：武器、防御、工程、工业、电子五类槽位。
- 插件必须先用材料制造出来，再从当前空间站本地库存安装到舰船。
- 新增材料链：钛合金、纳米陶瓷、超导纤维、量子透镜、等离子电容、制导芯片。
- 新增攻击方式：磁轨炮、脉冲激光、轻型导弹仓；导弹仓会真实消耗导弹弹药。
- 敌舰拥有独立护盾与装甲层；热能擅长破盾、动能擅长破甲、导弹具备范围爆炸伤害。
- 目标面板实时显示双层耐久、当前伤害效率和武器克制提示，命中时显示分层伤害数字。
- 空间站制造形成地理专精：赫利俄斯偏重工业、倪克斯偏晶体精炼、万塔偏军需生产，额外产物仍保留在当地仓库。
- 舰船状态移至画面底部中央：青色外环显示护盾，橙色内环显示装甲，操作按钮分列圆环两侧。
- 左上角大型舰船状态卡已移除，学院、合约和货舱信息整体上移；盾甲过低时圆环状态会自动警告。
- 市场加入 11 张统一风格的商品图片：列表使用缩略图快速识别，详情显示更大的选中商品图。
- 市场列表删除重复的库存与持仓说明，只保留商品名、价格和涨跌幅；完整说明改为悬停查看。
- 制作配方和插件蓝图加入“一键采购”：按本站实时价格购买全部缺失材料，含税总价直接显示在按钮上。
- 一键采购严格使用当前空间站库存，整单校验余额与库存，成功后材料进入本站独立仓库。
- 全局界面升级为深空版 iOS 风格：减少高亮描边和装饰小字，统一玻璃层级、18px 圆角与弹性按压。
- 空间站导航改用轻量胶囊选中状态，顶部功能入口隐藏重复说明，只保留图标与名称。
- 已安装武器与基础脉冲炮可使用 `Q` 或点击底部武器按钮实时切换。
- 武器加入电容与热量：激光耗电和发热最高，磁轨炮偏低耗稳定，导弹低耗但消耗实体弹药。
- 热量达到上限后武器进入强制冷却，降至安全值后自动解除锁定。
- 敌方侦察机、截击舰、导弹舰与重型炮舰分别使用不同伤害类型，盾甲克制关系同样作用于玩家。
- 插件会真实影响火力、护盾、装甲、速度、货舱、采矿效率、稀有矿概率和弹道表现。
- 市场、舰船维护、任务大厅、仓库货舱、星图导航、情报终端和系统设置都作为站内模块进入。
- 顶部状态会随模块切换显示“星港中枢 / 市场终端 / 舰船维护”等。
- 新增 2.5D 视觉层：空间站透视、模块悬停层次、星空视差和可关闭/标准/增强三档 3D 视觉设置。
- 玩家舰船升级为“拓荒者级”Canvas 模型，包含主艇身、收敛侧翼、双引擎、座舱、推进尾焰和护盾层。
- 新手引导升级为“边境飞行员执照”：教学停靠、采矿、本地交易、远程行情、独立仓库与自由合约。
- 合约中心新增“边境档案”主线面板，主线只提供目标与世界观方向，不直接改变价格、库存或税率。
